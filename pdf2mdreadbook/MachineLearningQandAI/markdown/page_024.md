### 1. Labeling More Data
**标注更多数据**
最直接的方案：人工标注更多数据，扩大标注集规模，提升模型泛化能力。

### 2. Bootstrapping the Data
**数据自助法**
通过有放回重采样生成多份训练数据，训练多个模型再集成，提升模型鲁棒性。

### 3. Transfer Learning
**迁移学习**
利用在大数据集上预训练好的模型，在小数据集上微调，复用通用知识，减少对标注数据的依赖。

### 4. Self-Supervised Learning
**自监督学习**
用数据自身构造监督信号（如掩码预测、对比学习），从无标注数据中学习通用特征，再适配下游小样本任务。

### 5. Active Learning
**主动学习**
让模型主动选择最有价值的样本交给人工标注，用最少的标注预算最大化模型收益。

### 6. Few-Shot Learning
**少样本学习**
让模型仅用少量标注样本就能快速学习新任务，如通过提示学习、元学习实现。

### 7. Meta-Learning
**元学习 / 学会学习**
让模型在大量不同任务上训练，学会快速适应新任务的“学习方法”，在小样本场景下快速收敛。

### 8. Weakly Supervised Learning
**弱监督学习**
利用不精确、不完整或间接的监督信号（如粗粒度标签、噪声标签）训练模型，降低对高质量标注的依赖。

### 9. Semi-Supervised Learning
**半监督学习**
同时使用少量标注数据和大量无标注数据训练模型，如通过伪标签、一致性正则化利用无标注数据的分布信息。

### 10. Self-Training
**自训练**
先用少量标注数据训练模型，再用模型对无标注数据生成伪标签，将高置信度样本加入训练集迭代优化。

### 11. Multi-Task Learning
**多任务学习**
让模型同时学习多个相关任务，共享底层特征，利用任务间的互补信息提升小样本任务的性能。

### 12. Multimodal Learning
**多模态学习**
融合文本、图像、音频等多种模态数据，利用不同模态的互补信息弥补标注数据的不足。

### 13. Inductive Biases
**归纳偏置**
通过模型结构、正则化、先验知识等方式引入额外约束，引导模型学习合理的泛化模式，减少对数据量的需求。
